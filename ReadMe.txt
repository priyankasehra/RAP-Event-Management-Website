STEPS TO RUN THE PROGRAM:

- Download XAMPP Control Panel.
- Put 'RAP' folder in "...xampp\htdocs" folder.
- Open XAMPP Control Panel and connect to Apache and MySQL(if you're facing any issue then try changing the port).
- On your browser go to "localhost/phpmyadmin/" and create a database then import the provided database "rap.sql" from sql folder.
- Check the mysql details in 'config.php' file.
- Put 'localhost/RAP/home-page.php' in the search bar of your browser to run the program.
